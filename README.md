# Alagamento de Silhuetas

## algoritmo para realizar o cálculo da quantidade de água que caberia se a despejarmos do topo das silhuetas

Para rodar o projeto basta ter o node instalado, e dentro da pasta rodar o comando `node index.js`
Caso, queira fazer os inputs manualmente, você pode abrir o arquivo `script-jsfiddle.js`, copiar o código e colar no js fiddle. Assim, conseguirá rodar o algoritmo e inserir as informações manualmente.

Foi realizado, ao todo, 17 testes unitários para validação das funções auxiliares, com 100% de coverage.

Ao rodar o projeto, os resultados serão apresentados no `log`. Foi-se utilizado o `console.log` para apresentação dos resultados.
